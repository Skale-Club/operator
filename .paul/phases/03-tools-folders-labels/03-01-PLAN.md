<objective>
## Goal
Add folder grouping and labels to the tools page.
- `folder`: a tool belongs to one folder (text, nullable). The table groups rows by folder.
- `labels`: a tool has zero or more labels (text[], nullable). Labels appear as small badges on each row.

## Purpose
Orgs with many tool configs need a way to organize them. Folders group related tools visually.
Labels add free-form tagging for filtering and quick recognition.

## Output
- Migration 016 adds `folder text` and `labels text[]` to `tool_configs`
- `database.ts` updated with new columns
- Actions updated to read/write folder and labels
- Table groups tools by folder (ungrouped tools in an "Other" section at bottom)
- Each tool row shows its labels as small colored badges
- Form has a Folder field (text input with autocomplete from existing folders) and a Labels field (tag input)
</objective>

    **Architecture change:** Instead of using react-table's grouping (complex), implement folder
    grouping manually in the render pass. Keep react-table for sorting/column defs but render
    rows grouped by folder.

    **Grouping logic:**
    ```ts
    // Group rows by folder
    const grouped = useMemo(() => {
      const map = new Map<string, ToolConfigWithIntegration[]>()
      for (const tool of toolConfigs) {
        const key = tool.folder ?? '__other__'
        if (!map.has(key)) map.set(key, [])
        map.get(key)!.push(tool)
      }
      // Sort: named folders alphabetically, then __other__ last
      const entries = [...map.entries()].sort(([a], [b]) => {
        if (a === '__other__') return 1
        if (b === '__other__') return -1
        return a.localeCompare(b)
      })
      return entries
    }, [toolConfigs])
    ```

    **Table structure:** Render one `<Table>` with the header once, then for each folder group:
    - A `<TableRow>` acting as a section header (colspan = all columns):
      ```tsx
      <TableRow className="bg-muted/30 hover:bg-muted/30">
        <TableCell colSpan={columns.length} className="py-1.5 px-4">
          <span className="text-xs font-medium text-muted-foreground uppercase tracking-wide">
            {folder === '__other__' ? 'Other' : folder}
          </span>
          <span className="ml-2 text-xs text-muted-foreground">({tools.length})</span>
        </TableCell>
      </TableRow>
      ```
    - Then the tool rows for that group

    **Labels field** — add after the folder field.
    Implement a simple tag input using local state (no extra library):
    - Controlled via `form.setValue('labels', [...])` 
    - Input captures text; on Enter key or comma key, trims and adds tag if non-empty and not duplicate
    - Tags displayed as chips with × button
    - Add `labelsInput` local state for the text input value

    ```tsx
    // Add to component local state:
    const [labelsInput, setLabelsInput] = useState('')
    const currentLabels: string[] = form.watch('labels') ?? []

    function addLabel(value: string) {
      const trimmed = value.trim().replace(/,$/, '')
      if (!trimmed || currentLabels.includes(trimmed)) return
      form.setValue('labels', [...currentLabels, trimmed])
      setLabelsInput('')
    }

    function removeLabel(label: string) {
      form.setValue('labels', currentLabels.filter((l) => l !== label))
    }
    ```

    ```tsx
    <FormItem>
      <FormLabel>Labels <span className="text-muted-foreground font-normal">(optional)</span></FormLabel>
      <div className="flex flex-wrap gap-1 mb-1">
        {currentLabels.map((label) => (
          <span key={label} className="inline-flex items-center gap-1 text-xs bg-secondary text-secondary-foreground px-2 py-0.5 rounded-full">
            {label}
            <button type="button" onClick={() => removeLabel(label)} className="hover:text-destructive">×</button>
          </span>
        ))}
      </div>
      <FormControl>
        <Input
          placeholder="Type label, press Enter"
          value={labelsInput}
          onChange={(e) => setLabelsInput(e.target.value)}
          onKeyDown={(e) => {
            if (e.key === 'Enter' || e.key === ',') {
              e.preventDefault()
              addLabel(labelsInput)
            }
          }}
          onBlur={() => { if (labelsInput) addLabel(labelsInput) }}
        />
      </FormControl>
      <FormMessage />
    </FormItem>
    ```

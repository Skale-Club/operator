# Plan 40-05: Agent Badge on Chat Messages

## Objective

Show a small agent badge on each assistant message in the chat-area inbox identifying which agent produced it. The badge reads the `agent_id` from `conversation_messages.metadata` (written by plan 40-01 Task 3). The agent name is resolved by passing an `agentMap` from the server to the message-list component.
